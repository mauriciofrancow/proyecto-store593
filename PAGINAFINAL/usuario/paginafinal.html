STORE 593 "THE BEST CLOTHING STORE"

Somos un grupo de estudiantes inspirados en lo mejor que tiene nuestro maravilloso ECUADOR🇪🇨 Nuestra tienda en línea está enfocada la reactivación de q ecuatorianos solucionando la falta de dinero para comenzar un emprendimiento enfocando el ámbito digital mediante una página web.

Este proyecto es hecho con html5, css y java script. Esta tienda digital ayudara a fábricas y pequeños emprendedores. La tienda virtual esta creada a base del conocimiento adquirido en estos últimos tres meses de clases. La página consta de una interfaz de usuario la cual nos muestra los productos que están en venta, una barra de búsqueda la cual nos permite verificar la existencia del producto. También la interfaz de registro la cual permitirá al usuario crearse una cuenta para realizar sus compras, también consta de una pestaña donde el usuario podrá ver su carrito de compras y el precio a pagar.
